Converting GENEA avatar from FBX to GLTF in Blender:
- Import avatar FBX
	- No Animation
- Correct Transforms
	- Select Object (Armature) > Object > Apply > All transforms
	- Select Mesh (LowP_01) > Object > Apply > All transforms
- Setup Texture
	- Outliner > Blender File
		- Delete all Materials
		- Delete all Images
	- Select Mesh (Outliner > View Layer)
	- Material > Add Material
	- Base Color > Image Texture > New
		- 1024px
		- 32-bit Float
	- Paint
- Export glTF
	- Remove all other objects in scene
	- Format > glTF Separate
	- Change output file name
